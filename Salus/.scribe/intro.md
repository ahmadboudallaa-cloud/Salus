# Introduction

API REST pour suivre les symptomes, prendre des rendez-vous et obtenir des conseils de bien-etre generes par IA.

<aside>
    <strong>Base URL</strong>: <code>http://localhost:8000</code>
</aside>

    <aside>Les routes protegees exigent un header Authorization avec un token Sanctum: <code>Authorization: Bearer {token}</code>.</aside>
